# 2D Zombie Survival (Top-Down)

Топ-даун шутер про выживание на Python + **pygame** с процедурной графикой (спрайты).

Вы играете за выжившего: вокруг спавнятся зомби, бегуны, громилы и **солдаты-стрелки** (стреляют в ответ). Стреляйте, собирайте опыт и лут, прокачивайте урон и HP. Можно входить в здания (внутри мебель и лут, снаружи всё скрыто) и **водить машины** — но бензин есть не у всех.

## Возможности

- 🧟 **Новые враги**: зомби, бегун, громила и солдат-стрелок (держит дистанцию и стреляет)
- 🔫 **Разное оружие**: пистолет (∶∞), автомат, дробовик, винтовка — у каждого свои патроны
- 🎒 **Инвентарь** (клавиша `I`): оружие, патроны и лут
- ⭐ **Сложность растёт**: чем дальше, тем больше опыта и лута падает с мобов; сильные враги дают больше
- 🚗 **Машины можно водить** (`E`): у части нет бензина; есть спидометр и индикатор топлива; можно давить зомби
- 🏃 **Скорость героя постоянная** — при прокачке растут только HP и урон
- 🏠 Здания с мебелью и лутом, 🌳 деревья, 🗺️ большая случайная карта

## Своя графика (необязательно)

Игра рисует спрайты кодом. Но если положить PNG в папку `assets/` (рядом с `main.py`), они будут использованы вместо них: `player.png`, `zombie.png`, `runner.png`, `brute.png`, `soldier.png`, `tree.png`, `item.png`, `ammo.png`, `orb.png`.

## Установка и запуск

```bash
pip install -r requirements.txt
python main.py
```

## Управление

| Действие | Клавиша |
|---|---|
| Движение / руль | `W` `A` `S` `D` или стрелки |
| Прицел / стрельба | Мышь / ЛКМ |
| Смена оружия | `1`–`4` или колёсико мыши |
| Инвентарь | `I` |
| Сесть / выйти из машины | `E` |
| Войти / выйти из здания | Подойти к двери / выходу |
| Рестарт / выход | `R` / `ESC` |

## Структура

```
main.py             # игровой цикл, рендер, UI, вождение
game/
  settings.py       # константы
  assets.py         # спрайты (+ загрузка PNG из assets/)
  physics.py        # коллизии
  camera.py         # камера
  player.py         # игрок (скорость не растёт)
  inventory.py      # инвентарь
  weapon.py         # оружие и патроны
  bullet.py         # пули (свои и вражеские)
  enemy.py          # враги (вкл. солдат-стрелок)
  pickup.py         # наземный лут / патроны
  world.py          # генерация мира + машины с бензином
  building.py       # здания и интерьеры
requirements.txt
```

Требуется Python 3.9+.
