# 2D Zombie Survival (Top-Down)

Топ-даун шутер про выживание, написанный на Python с использованием **pygame**.

Вы играете за выжившего: вокруг спавнятся зомби и другие сущности, вы отстреливаетесь, собираете опыт с убитых врагов и прокачиваете характеристики (урон / здоровье / скорость бега). По миру разбросаны деревья, автомобили и здания. В здания можно входить — внутри случайно сгенерированная комната с мебелью и предметами (лут), при этом внешний мир скрывается. Карта большая и генерируется случайно при каждом запуске.

## Возможности

- 🧟 Спавн зомби, бегунов и громил (brute) вокруг игрока, сложность растёт со временем
- 🔫 Стрельба мышкой, прицеливание по курсору
- ⭐ Сбор опыта с убитых врагов и автоматическая прокачка (урон / HP / скорость)
- 🏠 Здания, в которые можно войти — внутри мебель и предметы (лут), снаружи всё скрывается
- 🌳 Деревья, 🚗 автомобили и другие препятствия
- 🗺️ Большая случайно генерируемая карта при каждом запуске
- ❤️ Полоса здоровья, опыта, счёт, экран Game Over с рестартом

## Установка

```bash
pip install -r requirements.txt
```

## Запуск

```bash
python main.py
```

## Управление

| Действие | Клавиша |
|---|---|
| Движение | `W` `A` `S` `D` или стрелки |
| Прицеливание | Мышь |
| Стрельба | ЛКМ (зажать) |
| Войти в здание | Подойти к двери |
| Выйти из здания | Подойти к выходу (снизу) |
| Рестарт (после смерти) | `R` |
| Выход | `ESC` |

## Структура проекта

```
main.py             # точка входа и игровой цикл
game/
  settings.py       # константы и настройки
  physics.py        # коллизии
  camera.py         # камера
  player.py         # игрок, характеристики, прокачка
  bullet.py         # пули
  enemy.py          # враги (zombie / runner / brute)
  world.py          # генерация мира (деревья, авто, здания)
  building.py       # здания и интерьеры (мебель, лут)
requirements.txt
```

Требуется Python 3.9+.
